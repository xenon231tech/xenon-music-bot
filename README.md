# Xenon Music Bot 🎵

A Telegram bot that plays music directly from YouTube. Built with Python and yt-dlp.

## Features
- Play YouTube audio with `/play [song title]`
- Premium subscription system (Telegram Stars)
- Runs 24/7 on Android (Termux + PM2)

## Commands
- `/start` - Start the bot
- `/play [title]` - Play a song  
- `/stop` - Stop playback
- `/about` - Bot info
- `/premium` - Upgrade to premium
